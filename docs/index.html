Scoring System

Your performance is evaluated across three key dimensions

Empathy

Target: 75%+
  • Acknowledgment of patient concerns
  • Empathetic language and validation
  • Avoiding dismissive or condescending tone

Accuracy

Target: 75%+
  • Factual correctness of information
  • Evidence-based medical advice
  • Avoiding misinformation

Clarity

Target: 75%+
  • Message readability and structure
  • Optimal length (15-60 words)
  • Appropriate use of technical terms

Prerequisites

  • Python 3.8 or higher
  • pip (Python package manager)

Installation

# Clone the repository
git clone https://github.com/mcphersonlab/pharm-comm-ai.git
cd pharm-comm-ai

# Install dependencies
pip install -r requirements.txt

# Download NLTK data
python -c "import nltk; nltk.download('vader_lexicon')"

Run the Application

# Start the Flask server
cd backend
python app.py

# Open in browser
# Navigate to: http://localhost:5000

Architecture

Backend (Python/Flask)

  • Modular Design: Separated concerns for conversation simulation, NLP scoring, and feedback generation
  • Flask API: RESTful endpoints for session management and message processing
  • NLP Scoring: Uses TextBlob and NLTK for sentiment analysis and scoring
  • Conversation Simulator: Dynamic patient responses based on student performance
  • Feedback Generator: Provides detailed, actionable feedback

Frontend (Web-based)

  • Responsive Design: Works on desktop and mobile devices
  • Real-time Updates: Instant feedback display and score visualization
  • Interactive UI: Clean, professional interface for training sessions
  • Score Visualization: Progress bars and color-coded feedback
